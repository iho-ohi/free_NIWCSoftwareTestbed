# ShoreECIDS v2.0.0 Registration Dialog Button Fix

In this release of ShoreECDIS there is a known issue with the support text and continue button being cut off with text higher than 100%. This will be resolved as a patch v*2.0.1*.

# How to fix
If you notice the text or button cut off then follow these instructions.
 1. On windows searchbar type in `Settings`
 2. Click on `Display`
 3. From here scroll down to `Scale & layout`
 4. Under `Scale` slect from the dropdown the display size with (Recommended) next to it
 5. Under `Display resolution` slect from the dropdown the display resolution with (Recommended) next to it   


 6. Navigate back to `Settings`
 7. Click on `Accessiblity`, it is the third option from the bottom
 8. From here navigate to `Vision` then click `Text size` 
 9. Once in `Text size` under the sub field `Text size`, make sure the slider is all the way down at 100%

# Alternate Fix
If these steps do not fix the dialog issues then it is possible to highlight the continue btton by pressing the `tab` button on your keyboard until the continue button is highlighted (seven clicks) from there pressing `enter` will confirm the registration dialog.
